# مترجم الأوامر العربية إلى Blender
# Arabic Blender Translator

## 🎯 الميزات الرئيسية

✅ **بدون API Keys** - استخدام Windows AI المدمج
✅ **واجهة رسومية احترافية** - سهلة وجميلة
✅ **دعم كامل للعربية** - جميع الأوامر والألوان والخصائص
✅ **اتصال فوري بـ Blender** - TCP Socket مباشر
✅ **توليد كود Python** - احترافي وقابل للتعديل
✅ **سجل الأوامر** - تتبع الأوامر السابقة

---

## 📦 المكونات

### 1. `nlp_processor.py`
معالج اللغة الطبيعية للعربية
- تحليل الأوامر العربية
- استخراج الألوان والمواضع والخصائص
- توليد أوامر Blender

### 2. `gui_pro.py`
الواجهة الرسومية المحترفة
- إدخال الأوامر بالعربية
- عرض الكود المولد
- سجل الأوامر والتاريخ
- اتصال فوري بـ Blender

### 3. `blender_addon_pro.py`
إضافة Blender
- خادم TCP يستقبل الأوامر
- تنفيذ الكود مباشرة
- تسجيل الأحداث

---

## 🚀 التثبيت والاستخدام

### خطوة 1: تثبيت الملفات
```bash
git clone https://github.com/ahmed25553/blender-arabic-translator.git
cd blender-arabic-translator
```

### خطوة 2: تثبيت الإضافة في Blender
1. افتح Blender
2. اذهب إلى Edit > Preferences > Add-ons
3. اضغط Install
4. اختر `blender_addon_pro.py`
5. فعّل الإضافة

### خطوة 3: تشغيل التطبيق الرسومي
```bash
python gui_pro.py
```

## 📝 أمثلة الأوامر

### إنشاء أشكال
- `مكعب أحمر` → مكعب لون أحمر
- `كرة زرقاء كبيرة` → كرة زرقاء حجم كبير
- `اسطوانة خضراء` → اسطوانة لون أخضر
- `مخروط أصفر` → مخروط لون أصفر

### التحريك والتموضع
- `مكعب أحمر يمين` → مكعب أحمر موضوع على اليمين
- `كرة أزرق أعلى` → كرة زرقاء فوق المركز
- `اسطوانة أسفل` → اسطوانة تحت المركز

### التعديلات
- `احذف` → حذف الكائن المحدد
- `نسخ` → نسخ الكائن المحدد
- `اخف` → إخفاء الكائن
- `اظهر` → إظهار الكائن

## 🔌 التكامل مع Blender

البرنامج يستخدم **TCP Socket** للتواصل:

```
التطبيق الرسومي
    ↓ (TCP على المنفذ 9998)
خادم Blender
    ↓
تنفيذ الكود
    ↓
النتيجة مباشرة 🎨
```

## 💡 نصائح الاستخدام

## 🐛 استكشاف الأعطال

### "غير متصل - افتح Blender"
- تأكد من فتح Blender
- تأكد من تثبيت الإضافة
- أعد تشغيل Blender

### "لم يتم التعرف على الأمر"
- تحقق من تهجئة الأمر
- استخدم الأوامر المدعومة من القائمة
- راجع القاموس أعلاه

### "فشل تنفيذ الكود"
- تحقق من أن الكائن مختار
- راجع رسالة الخطأ في منطقة المعلومات
- جرب أمر أبسط
