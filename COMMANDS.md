# قاموس الأوامر المحسّن
# Advanced Commands Dictionary

## 🎯 الأوامر الأساسية

### إنشاء الأشكال (Create Shapes)

#### مكعبات
```
مكعب                    → مكعب عادي
مكعب أحمر               → مكعب أحمر اللون
مكعب أحمر كبير          → مكعب أحمر حجم كبير
مكعب أحمر يمين          → مكعب أحمر على اليمين
مكعب أزرق كبير أعلى     → مكعب أزرق كبير فوق المركز
```

#### كرات
```
كرة                     → كرة عادية
كرة زرقاء               → كرة زرقاء اللون
كرة زرقاء كبيرة         → كرة زرقاء كبيرة الحجم
كرة خضراء صغيرة         → كرة خضراء صغيرة الحجم
```

#### اسطوانات
```
اسطوانة                 → اسطوانة عادية
اسطوانة خضراء           → اسطوانة خضراء
اسطوانة أصفر كبيرة      → اسطوانة صفراء كبيرة
```

#### مخاريط
```
مخروط                   → مخروط عادي
مخروط برتقالي           → مخروط برتقالي اللون
مخروط أسود ضخم          → مخروط أسود كبير جداً
```

#### حلقات (Torus)
```
تورس                    → حلقة عادية
حلقة بنفسجية            → حلقة بنفسجية
تورس أبيض كبير          → حلقة بيضاء كبيرة
```

## 📍 المواضع والاتجاهات (Positions)

```
يمين       → تحريك 2 وحدة لليمين (X+)
يسار       → تحريك 2 وحدة لليسار (X-)
أمام       → تحريك 2 وحدة لأمام (Y+)
خلف       → تحريك 2 وحدة للخلف (Y-)
أعلى       → تحريك 2 وحدة لأعلى (Z+)
أسفل       → تحريك 2 وحدة لأسفل (Z-)
```

**أمثلة:**
```
مكعب أحمر يمين         → مكعب أحمر على اليمين
كرة زرقاء أعلى         → كرة زرقاء فوق المركز
اسطوانة أسفل           → اسطوانة تحت المركز
```

## 📏 أحجام وخصائص (Sizes & Properties)

### الأحجام
```
كبير      → حجم x2 (ضعف الحجم العادي)
صغير      → حجم x0.5 (نصف الحجم)
ضخم       → حجم x3 (ثلاثة أضعاف)
دقيق      → حجم x0.3 (صغير جداً)
```

### الخصائص المرئية
```
شفاف       → 50% شفافية
نصف شفاف   → 30% شفافية
معدني      → مادة معدنية لامعة
لامع       → مادة لامعة جداً
غير لامع   → مادة غير لامعة (مطفية)
```

## 🔧 أوامر التحرير والتعديل (Edit Commands)

```
احذف       → حذف الكائن المحدد
امسح       → حذف الكائن (نفس احذف)
نسخ        → نسخ الكائن المحدد
انسخ       → نسخ الكائن (نفس نسخ)
الصق       → لصق الكائن المنسوخ
اخف        → إخفاء الكائن من العرض
اظهر       → إظهار الكائن المخفي
جمّع       → تجميع عدة كائنات
فك         → فك التجميع
```

## 🎬 أمثلة عملية متقدمة (Advanced Examples)

### مشهد بسيط
```
مكعب أحمر يمين
كرة زرقاء يسار
اسطوانة خضراء أعلى
```

### تعديل الكائنات
```
مكعب أحمر كبير ضخم       → مكعب أحمر حقاً كبير!
كرة زرقاء صغير جداً      → كرة زرقاء دقيقة
مخروط بنفسجي معدني       → مخروط معدني لامع
```

### دمج الخصائص
```
مكعب + لون + حجم + موضع
مثال: كرة برتقالي ضخم أعلى يمين
```

## ⚡ النصائح السريعة (Quick Tips)

✅ **الترتيب غير مهم:**
```
مكعب أحمر يمين = يمين أحمر مكعب = أحمر مكعب يمين
```

✅ **يمكن مزج المرادفا��:**
```
مكعب = مكعب سداسي = كيوب
احذف = امسح = استبدل
```

✅ **أكثر من خاصية:**
```
مكعب أحمر كبير معدني شفاف
```

⚠️ **قيود:**
- كل أمر ينفذ على الكائن المحدد
- تأكد من تحديد كائن قبل احذف
- الألوان مدمجة في النظام
