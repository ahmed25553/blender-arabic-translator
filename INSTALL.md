# Arabic Blender Translator - Installation & Setup Guide
# دليل التثبيت والإعداد - مترجم الأوامر العربية إلى Blender

## ⚡ البدء السريع (Quick Start)

### على Windows:
```bash
1. افتح المجلد
2. انقر مرتين على run.bat
3. اتبع التعليمات
```

### على Linux/Mac:
```bash
chmod +x run.sh
./run.sh
```

---

## 📋 المتطلبات الأساسية

✅ **Python 3.7+**
- تنزيل: https://www.python.org/downloads/
- تأكد من تحديد "Add Python to PATH" أثناء التثبيت

✅ **Blender 3.0+**
- تنزيل: https://www.blender.org/download/
- أي نسخة حديثة تعمل

✅ **Windows 10/11 أو Linux أو Mac**

---

## 🔧 خطوات التثبيت الكاملة

### الخطوة 1: تنزيل المستودع
```bash
git clone https://github.com/ahmed25553/blender-arabic-translator.git
cd blender-arabic-translator
```

أو انقر على `Code > Download ZIP` وفك الضغط

### الخطوة 2: تثبيت الإضافة في Blender

1. **افتح Blender**

2. **اذهب إلى الإعدادات**
   - Edit → Preferences (أو Ctrl+,)

3. **افتح قسم Add-ons**
   - اضغط على "Add-ons" من الشريط الجانبي

4. **اضغط Install**
   - في أعلى الجانب الأيمن

5. **اختر الملف**
   - ابحث عن `blender_addon_pro.py`
   - اختره وأكمل

6. **فعّل الإضافة**
   - ابحث عن "Arabic Blender Translator Pro"
   - اضغط على الصندوق لتفعيلها
   - يجب أن ترى علامة صح ✅

7. **احفظ التفضيلات**
   - Ctrl+U أو Save Preferences

### الخطوة 3: تشغيل التطبيق الرسومي

**على Windows:**
```bash
run.bat
```

**على Linux/Mac:**
```bash
./run.sh
```

**أو مباشرة:**
```bash
python gui_pro.py
```

## ✅ التحقق من التثبيت

### هل الإضافة مثبتة بشكل صحيح؟
1. افتح Blender
2. انظر إلى الجانب الأيمن (View3D)
3. ابحث عن تبويب "Arabic"
4. يجب أن ترى "مترجم الأوامر العربية"

### هل الخادم يعمل؟
1. شغّل البرنامج الرسومي `gui_pro.py`
2. انظر إلى الأعلى
3. يجب أن تكون الحالة **🟢 متصل بـ Blender**

---

## 🎯 أول أمر تجريبي

1. **افتح Blender** وتأكد من فتح التطبيق الرسومي
2. **في التطبيق الرسومي، اكتب:**
   ```
   مكعب أحمر
   ```
3. **اضغط Enter** أو اضغط على **▶️ تنفيذ الأمر**

4. **انظر إلى Blender** - يجب أن تظهر مكعب أحمر! 🎨

## 🛠️ استكشاف الأعطال

### ❌ "غير متصل - افتح Blender"

**الحل:**
1. تأكد من فتح Blender
2. تأكد من تثبيت الإضافة وتفعيلها
3. أعد تشغيل Blender
4. الإضافة تحتاج وقت للتحميل - انتظر 5 ثواني

### ❌ "فشل تنفيذ الأمر"

**الحل:**
1. تأكد من تحديد كائن في Blender (مثل المكعب الافتراضي)
2. جرب أمر بسيط: `احذف`
3. تحقق من رسالة الخطأ في منطقة "التفاصيل"

### ❌ "Error: No module named 'tkinter'"

**الحل (Windows):**
```bash
python -m pip install tk
```

**الحل (Linux - Ubuntu/Debian):**
```bash
sudo apt-get install python3-tk
```

**الحل (Mac):**
```bash
brew install python-tk
```

### ❌ "لم يتم التعرف على الأمر"

**الحل:**
1. استخدم الأوامر من الأمثلة السريعة
2. تحقق من القاموس في README.md
3. تأكد من تهجئة الأمر بالعربية

## 📞 الدعم والمساعدة

### مشاكل شائعة:
1. **Blender لم يستجب؟**
   - أعد تشغيل Blender
   - تحقق من تفعيل الإضافة

2. **البرنامج الرسومي لا يفتح؟**
   - تحقق من Python: `python --version`

3. **الكود لا ينفذ؟**
   - تحقق من أن الكائن مختار في Blender
   - اقرأ رسالة الخطأ في "التفاصيل"
